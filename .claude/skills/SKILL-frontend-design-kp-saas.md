# Frontend Design — КП-генератор для российского B2B

## Контекст продукта

Целевая аудитория: фрилансеры, ИП, малые агентства в России (35-55 лет).
Психология: ценят практичность, доверие, скорость. Не доверяют "гладким" западным сайтам.
Задача дизайна: создать ощущение "это сделано для меня, не для американцев".

---

## Дизайн-мышление для этого продукта

**Тон:** Деловой без официоза. Как хороший менеджер по продажам — уверенный, конкретный, без воды.

**Эстетика:** НЕ стартап-стерильность. НЕ purple gradients. 
Выбери одно из: 
- "Строгий профессионал" — тёмно-синий + белый + точечный акцент
- "Рабочий инструмент" — нейтральные серые + яркий CTA
- "Современный русский бизнес" — глубокий зелёный/синий + тёплые бежевые

**Запоминаемость:** Цифра "30 секунд" должна быть визуально центральной. Это главный крючок.

## Обязательные паттерны для каждой страницы

### Страница / (Лендинг + Форма)

```
СТРУКТУРА:
1. Hero: большая цифра "30" или таймер как визуальный акцент
2. Форма: 6 полей — максимально простые лейблы, русский текст плейсхолдеров
3. Счётчик КП: заметный, не назойливый — "Осталось X из 3 бесплатных"
4. Социальное доказательство: даже если заглушка — конкретные цифры

ТИПОГРАФИКА:
- Заголовок: крупный, жирный, с переносом на ключевом слове
- Подзаголовок: меньше, с акцентом на конкурентах ("пока конкуренты тратят 3 часа")
- Поля формы: 16px минимум, достаточный line-height — читабельность важнее компактности

ЦВЕТА:
- Primary: #1e3a5f (тёмно-синий) или #064e3b (глубокий зелёный)
- Accent: #f59e0b (янтарный) — только для CTA и ключевых цифр
- Background: #f8fafc (почти белый, не слепящий)
- Text: #1e293b (не чёрный — мягче)

КНОПКА "СОЗДАТЬ КП":
- Полная ширина формы
- Высота 56px минимум (мобайл-first)
- Hover: небольшое поднятие + shadow
- Loading state: спиннер + "Генерирую КП..."

МОБИЛЬНАЯ ВЕРСИЯ:
- Поля в одну колонку
- Sticky кнопка если форма длинная
- Счётчик — над кнопкой, не под
```

**Пример запроса к Claude Code:**
```
Создай лендинг страницу для КП-генератора.
Используй skill: frontend-design-kp-saas.
Стек: Next.js 14, Tailwind CSS, TypeScript.
Компонент: app/page.tsx
Форма с 6 полями (названия в промте), кнопка, счётчик.
Анимация: форма появляется снизу при загрузке (staggered, 6 полей по 80ms задержки).
```

### Страница /result (Предпросмотр КП)

```
СТРУКТУРА:
1. Превью КП: как настоящий документ (белая "бумага" с тенью на сером фоне)
2. Переключатель шаблонов: 3 миниатюры, выбранный — обведён акцентом
3. Панель действий: Скачать / Редактировать / Создать новое
4. Апсейл-баннер (если лимит): не агрессивный, но заметный

ПРЕВЬЮ ДОКУМЕНТА:
- Белый блок: max-width 680px, padding 48px, shadow-xl
- Типографика: serif для заголовка КП, sans для тела
- Имитация PDF: правильные отступы, разрывы абзацев

ПЕРЕКЛЮЧАТЕЛЬ ШАБЛОНОВ:
- Горизонтальный ряд из 3 карточек 80x110px
- Каждая показывает минипревью цветовой схемы
- "Классика" | "Тёмный" | "Минимализм"

КНОПКА СКАЧАТЬ:
- Зелёная (#10b981) — психология "готово, можно отправить"
- Иконка PDF рядом
```

**Пример запроса:**
```
Создай страницу предпросмотра результата КП.
Используй skill: frontend-design-kp-saas.
Компонент: app/result/page.tsx
Показывает: текст КП в виде "документа", 3 шаблона на выбор, кнопка скачать PDF.
Апсейл баннер снизу если проп showUpsell=true.
```

### Страница /dashboard (Личный кабинет)

```
СТРУКТУРА:
1. Шапка: имя пользователя, текущий тариф, счётчик
2. Список КП: таблица или карточки, дата + клиент + шаблон + скачать
3. Пустое состояние: если КП ещё нет — направляем на создание

ТАБЛИЦА КП:
- Не перегружать — дата, клиент, действия
- Hover row: лёгкое выделение
- Мобайл: карточки вместо таблицы

БЛОК ТАРИФА:
- Визуальный прогресс-бар: X из 3 КП использовано
- Если лимит — апсейл без агрессии
```

## Антипаттерны для ЭТОГО проекта

**Никогда не делай:**
- Gradient от фиолетового к синему — слишком "AI стартап"
- Emoji в заголовках — аудитория воспринимает как несерьёзность
- Анимированный background с частицами — замедляет и раздражает
- Sans-serif с thin weight для основного текста — плохо читается
- Hero с человеком за ноутбуком — stock photo убивает доверие

**Вместо этого:**
- Типографические акценты (большие цифры, цитаты)
- Иконки из lucide-react (строгие, не мультяшные)
- Реальные примеры текста КП в превью
- Тёплые нейтральные фоны (не холодные серые)

## Шрифтовые пары для этого проекта

**Вариант A (строгий профессионал):**
- Заголовки: `Playfair Display` (serif, authority)
- Тело: `Source Sans 3` (читабельный, нейтральный)

**Вариант B (современный инструмент):**
- Заголовки: `Geologica` (геометрический, современный)
- Тело: `Nunito Sans` (мягкий, дружелюбный)

**Вариант C (минималист):**
- Всё: `PT Serif` + `PT Sans` (русское происхождение = психологическое доверие)

Импорт через Google Fonts в `app/layout.tsx`.
