# КП за 30 секунд — Мастер-скилл разработчика

## Контекст продукта

**Продукт:** Веб-сервис генерации коммерческих предложений на базе ИИ.  
**Аудитория:** Фрилансеры, ИП, малые агентства (дизайн, SMM, разработка, консалтинг, юруслуги).  
**Ценность:** Пользователь заполняет 6 полей → получает готовое КП в PDF за 30 секунд.  
**Монетизация:** Freemium — 3 КП бесплатно → подписка 990 ₽/месяц.  
**Рынок:** Россия, 12+ млн самозанятых, рынок GenAI ×3 за 2025 год.

### Стек (предполагаемый, уточняй у пользователя)
- Backend: Python / Node.js
- LLM: OpenAI API или совместимый (Claude, YandexGPT)
- PDF: ReportLab (Python) или pdf-lib (JS)
- БД: PostgreSQL
- Голос: OpenAI Whisper API ($0.006/мин)
- Frontend: React / Next.js

## 15 улучшений: реестр и статус

Каждое улучшение имеет ID. При реализации всегда ссылайся на ID в комментариях кода.

### БЛОК 1 — Монетизация и конверсия

#### [F01] Тариф «Разово» — 149–199 ₽ за одно КП
**Цель:** Конвертировать пользователей, не готовых к подписке.  
**Логика:** После 3 бесплатных КП показываем выбор: «Разовая покупка 199 ₽» или «Подписка 990 ₽/мес».  
**Реализация:**
- Добавить `plan_type: enum('free', 'one_time', 'monthly', 'yearly')` в таблицу users
- Добавить `one_time_credits: int` — счётчик купленных разовых КП
- Платёжный экран: две кнопки с явным сравнением ценности
- После оплаты: `one_time_credits += 1`, списывать при генерации
- **Файл:** `references/billing.md` — полная схема тарифов

#### [F02] Годовая подписка — 7 990 ₽/год
**Цель:** Снизить churn, получить предоплаченный кэш.  
**Логика:** Скидка ~33% vs месячная. На экране оплаты показывать «Экономия 1 890 ₽».  
**Реализация:**
- Добавить `plan_period: enum('monthly', 'yearly')` 
- `subscription_ends_at: timestamp` вместо ежемесячного биллинга
- Баннер «Перейти на годовой» для активных месячных подписчиков через 14 дней

#### [F03] Тариф Pro — 1 990 ₽/месяц
**Цель:** Монетизировать агентства с повышенным ARPU.  
**Дополнительные функции Pro:**
- Загрузка логотипа и фирменных цветов в PDF
- 5 стилей оформления КП (минимализм, корпоративный, креативный, юридический, технический)
- Экспорт в DOCX + PDF
- Трекер статусов КП (отправлено / просмотрено / принято)
- Безлимитные КП
**Реализация:** Добавить `plan_tier: enum('free', 'basic', 'pro')`, проверки в генераторе

#### [F04] Партнёрская программа — 25% от оплаты реферала
**Цель:** Превратить каждого платного пользователя в канал привлечения.  
**Логика:** Реферальная ссылка в личном кабинете. Начисление при первой оплате реферала.  
**Реализация:**
- Таблица `referrals(referrer_id, referee_id, status, amount, paid_at)`
- UTM-параметр `ref=USER_ID` в реферальной ссылке
- Автоматическое начисление бонуса на баланс
- Минимальная сумма для вывода: 500 ₽

### БЛОК 2 — Удержание и снижение оттока

#### [F05] Триггерные уведомления
**Цель:** Возвращать забывших пользователей до отписки.  
**Триггеры (реализовать как cron + очередь):**
```
- через 7 дней без КП → email/TG: «Клиент ждёт ответа?»
- через 14 дней без КП → email: кейс успешной сделки
- за 3 дня до окончания подписки → предложение продлить
- после 1-го КП в день → поздравление + совет по отправке
```
**Стек:** Redis Queue / Celery + SMTP + Telegram Bot API

#### [F06] Личный кабинет с историей и статусами КП
**Цель:** Встроиться в рабочий процесс, стать незаменимым инструментом.  
**Функции:**
- Список всех КП с датой, клиентом, суммой
- Статус: Черновик / Отправлено / Просмотрено / Принято / Отклонено
- Дублирование КП (взять за основу прошлое)
- Поиск по клиенту или услуге
**Схема БД:**
```sql
CREATE TABLE proposals (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  client_name VARCHAR(255),
  service VARCHAR(255),
  price DECIMAL(12,2),
  status VARCHAR(50) DEFAULT 'draft',
  pdf_url TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

#### [F07] Онбординг: довести до первого КП за 3 минуты
**Цель:** Максимально быстрый wow-момент = выше конверсия в платных.  
**Шаги онбординга:**
1. Регистрация: только email + пароль (никаких лишних полей)
2. Приветственный экран: «Давайте создадим ваше первое КП прямо сейчас»
3. Предзаполненная форма с примером (пользователь меняет под себя)
4. Генерация → PDF → кнопка «Отправить на email» или «Скачать»
5. После скачивания: «Понравилось? Создавайте сколько угодно — 990 ₽/мес»

**Метрика:** % пользователей, создавших КП в первые 10 минут после регистрации. Цель: >60%.

#### [F08] Telegram-бот
**Цель:** Снизить трение — генерировать КП без входа на сайт.  
**Команды бота:**
```
/start — приветствие + авторизация
/kp — начать новое КП (пошаговый диалог)
/history — последние 5 КП
/status — статус подписки
/help — справка
```
**Авторизация:** Пользователь нажимает «Подключить Telegram» в ЛК → получает одноразовый код → вводит в бот → аккаунты связаны.  
**Стек:** python-telegram-bot или Aiogram 3.x

### БЛОК 4 — Продукт и доверие

#### [F13] Социальное доказательство
**Цель:** Снять возражение «а вдруг фигня» у новых пользователей.  
**Элементы:**
- Живой счётчик: «Уже создано X коммерческих предложений»
- 3–5 отзывов с именем, профессией, результатом
- Логотипы компаний-клиентов (если есть)
**Реализация:** Счётчик из БД (`SELECT COUNT(*) FROM proposals`), кешировать раз в 5 минут.

#### [F14] A/B тест: демо на лендинге без регистрации
**Цель:** Повысить конверсию лендинга показом результата до регистрации.  
**Описание:** Два поля на главной странице («Ваша услуга» + «Тип клиента») → кнопка «Показать пример» → стримится первый абзац КП → CTA «Получить полное КП бесплатно».  
**Ограничение:** 1 демо-запрос с IP, потом только после регистрации.  
**Подробная архитектура:** `references/ab-landing.md`

#### [F15] Расширение: договор, акт, счёт
**Цель:** Повысить ARPU без новых пользователей.  
**Новые типы документов:**
- Договор оказания услуг
- Акт выполненных работ  
- Счёт на оплату
**Реализация:** Новый тип в поле `document_type`, отдельные промпты и PDF-шаблоны.

## Голосовой ввод — ключевая архитектура

Полные детали в `references/voice-input.md`. Краткая схема:

```
Пользователь говорит
        ↓
[Web Audio API / MediaRecorder]  ← браузер/мобильный
        ↓
[Whisper API]  ← $0.006/мин, ~2-3 мин речи = ~1.5 ₽
        ↓
[NLU-слой: LLM-промпт структурирует текст по 6 полям КП]
        ↓
[Форма заполняется автоматически]  ← пользователь проверяет
        ↓
[Стандартный генератор КП]
        ↓
[PDF]
```

**Критические требования:**
- Всегда показывать распознанный текст пользователю до генерации КП
- Поддерживать ручную правку полей после распознавания
- Graceful degradation: если Whisper недоступен — показывать обычную форму
